Co je PHP?

  • Programovací jazyk a jeho běhové prostředí
  • Open source
  • Dostupný pro většinu platforem (Windows, Linux, Mac)
  • Cca od roku 1994
Začalo jako doplněk pro jednoduché dynamické stránky, časem se stal velmi oblíbeným a používaným jazykem.

Výhody PHP

  • Jednoduché a rychlé programování
  • Nekompiluje se, program se jen spustí
  • Dobrá podpora webhostingů a rozšířenost
  • Velká komunita, hodně zdrojů
  • Skvělá dokumentace
  • Neustále se zlepšující jazyk a ekosystém
  • Po PHP programátorech je poptávka!
  • Po programátorkách větší!:)

Nevýhody PHP

  • Nekonzistence v chování funkcí
  • Nekonzistence v pojmenování a parametrech funkcí
  • S každým požadavkem se provádí celý kód znovu
  • Podpora Unicode v základu
  • Začátečník rychle sklouzne ke špatnému kódu

Většina problémů se postupně řeší, každá nová verze PHP je o kus lepší.

Command line programy

Command line = příkazová řádka

  • PHP program se spouští z příkazové řádky
  • Vhodné například pro automatizaci úkolů

PHP prakticky

Jak se PHP spouští?

  • Musí být nainstalované na počítači/serveru
  • Přes web server nebo příkazovou řádku se zavolá soubor ve kterém je PHP uloženo:
    • http://www.stranka.cz/soubor.php
    • C:\Program Files\php\php.exe soubor.php

Jak vypadá PHP program?

Program PHP se ukládá do souboru s koncovkou .php

Soubor s PHP může obsahovat i jiný obsah než PHP, například HTML kód

Aby bylo PHP zpracováno, musí být uzavřeno mezi značkami <?php a ?>
