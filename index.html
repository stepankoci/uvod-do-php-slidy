Co je PHP?

  • Programovací jazyk a jeho běhové prostředí
  • Open source
  • Dostupný pro Windows, Linux, Mac atp.
  • Cca od roku 1994

Začalo jako doplněk pro jednoduché dynamické stránky, časem se stal velmi oblíbeným a používaným jazykem.

Výhody PHP

  • Jednoduché a rychlé programování
  • Nekompiluje se, program se jen spustí
  • Dobrá podpora webhostingů a rozšířenost
  • Velká komunita, hodně článků, návodů apod.
  • Skvělá oficiální dokumentace php.net
  • Neustále se zlepšující jazyk, knihovny a nástroje
  • Po PHP programátorech je poptávka!
  • Po programátorkách větší! :)

Nevýhody PHP

  • Nekonzistence v chování funkcí
  • Nekonzistence v pojmenování a parametrech funkcí
  • S každým požadavkem se provádí celý kód znovu
  • Není standardně podpora Unicode ve funkcích
  • Začátečník rychle sklouzne ke špatnému kódu

Většina problémů se postupně řeší, každá nová verze PHP je o kus lepší.

Co se dá s PHP dělat?

  • Programovat www stránky
  • Programovat www aplikace
  • Programovat command line programy
Další příklad třeba Imatic vstupní systém.
:)

www stránky

  • Chceme zobrazovat dynamický obsah
  • Web je rozsáhlý a nechceme spravovat desítky statických stránek
  • Používáme on-line administrační rozhraní
Příklady použití. CMS (Wordpress), drobnosti (zobrazení data, náhodné fotky, komentáře, ..)

www aplikace

  • V PHP můžeme psát od malých aplikací po velké systémy
  • E-shopy, informační systémy, rezervační systémy, sociální sítě...
Příklady použití: PHP čte data z databáze a zobrazuje je na webu. Uživatel přes webovou stránku ukládá data přes PHP do databáze. PHP je spouštěno na serveru a něco tam dělá (aktualizuje data podle jiné databáze, počítá...)

Command line programy

Command line = příkazová řádka

  • PHP program se spouští z příkazové řádky
  • Vhodné například pro zjednodušení a zrychlení rutinních úkolů
Např.: Hromadné přejmenování souborů. Zmenšení fotek z dovolené. Přesun/rozdělení fotek podle data vyfocení do různých adresářů. Hlídání slev v různých obchodech:)

A teď praxe...

Jak napsat první PHP program?

  • Vytvoříme například na disku C adresář php (přímo v C)
  • Vytvoříme v něm textový soubor test.php
  • Do souboru napíšeme: 
    <?php phpinfo(); ?>

Použijeme vestavěný webserver v PHP

Píšeme v poznámkovém bloku nebo jiném editoru, který máme, ne word:)

Jak spustit PHP program?

  • Otevřeme příkazovou řádku
    (stisknout WIN+R, napsat cmd a stisknout ENTER)
  • Do ní napíšeme cd \php
  • Spustíme webserver příkazem
    php -S localhost:8080 -t .
  • V prohlížeči otevřeme adresu
    localhost:8080/test.php
